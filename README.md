# Chess
 
vanilla JS chess engine with a minimax and alpha-beta pruning
 
**Play:** https://sriramyg.github.io/CHESS/
 
click somewhere on the board to start
 
Press `z` to undo.
 
## Files
 
- `main.js` — canvas rendering, click handling
- `gameState.js` — board state, move rules, check/castling/promotion
- `move.js` — move representation
- `engine.js` — engine: material + piece-square eval, minimax with alpha-beta pruning (depth 3-4)
